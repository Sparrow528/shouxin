# 守信出海 · 中国企业出海第一站

出海投资内容平台，提供：

- **每日出海早报** — 出海新闻速递
- **出海工具合集** — ODI备案/跨境财税/架构设计/地区指南
- **每日一工具** — 每天一个出海工具用法说明

## 部署到 GitHub Pages

1. 在 GitHub 上创建仓库（如 `shouxin`）
2. 将此目录下所有文件 push 到仓库
3. 在仓库 Settings → Pages 中，Source 选 `main`，文件夹选 `/ (root)`
4. 访问 `https://[你的用户名].github.io/[仓库名]/`

## 目录结构

```
├── index.html              # 首页（发朋友圈的入口）
├── 出海工具合集.html        # 工具合集（12个工具）
├── 出海早报_2026-06-16.html # 每日出海早报
└── (后续每日生成的早报也放这里)
```
