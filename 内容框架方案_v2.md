## 五、视觉风格系统

### 5.1 色彩体系

| 用途 | 色值 | 说明 |
|------|------|------|
| 主背景 | `#0F172A` | 深蓝，header/卡片底色 |
| 次背景 | `#1E293B` | 稍浅深蓝，渐变过渡 |
| 页面背景 | `#F8FAFC` | 浅灰白，保持清爽 |
| 主金色 | `#C8A84E` | 品牌金，按钮/标签/强调 |
| 文字白 | `#FFFFFF` | header文字 |
| 正文 | `#1E293B` | 卡片内标题 |
| 辅助文字 | `#64748B` | 描述性文字 |
| 分割线 | `#E2E8F0` | 浅灰分割 |
| 成功绿 | `#10B981` | 案例中的正面指标 |
| 警告橙 | `#F59E0B` | 案例中的风险/教训 |

### 5.2 字体层级

```
移动端（375px基准）:
  h1 页面大标题：28px / 800 weight
  h2 卡片标题：  18px / 700 weight
  h3 区块标题：  16px / 600 weight
  正文：        14px / 400 weight
  辅助文字：    12px / 400 weight
  标签：        10px / 600 weight
  
  line-height: 1.6-1.8（中文阅读友好）
  letter-spacing: 0.5-1px（标题微调）
```

### 5.3 卡片设计规范

```
基础卡片：
  border-radius: 14px
  padding: 20px
  background: #FFFFFF
  box-shadow: 0 1px 4px rgba(0,0,0,0.06)
  border: 1px solid rgba(0,0,0,0.04)
  
悬停/按压态（移动端无hover，用active）：
  transform: scale(0.97)
  transition: all 0.15s ease
  
金色标签：
  display: inline-block
  font-size: 10px / 600 weight
  color: #FFFFFF
  background: #C8A84E
  border-radius: 4px
  padding: 2px 8px
```
